The script gives as an output a .pdf file displaying a behaviour typical of a resonate and fire neuron, thanks to a Brian 2 simulation showing the response of the membrane voltage to a spike train.

Brian 2 will need to be implemented on the user's computer. More information on Brian 2 can be found in the README file at the root of the repository.

The user can either use the 'environment.yml' file which will take care of Brian 2 automatically, or, if they want to install the packages one by one, go for a manual install, through
```bash
pip install brian2
```
or
```bash
conda install -c conda-forge brian2
```
